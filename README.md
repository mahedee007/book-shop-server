# Book-shop-server
### **Objective **
Develop an Express application with TypeScript, integrating MongoDB with Mongoose to manage a Book Store. Ensure data integrity using Mongoose schema validation.


### Backend technologies Used in this project
- nodeJs
- expressJs
- mongoDb
- Mongoose
- typescript
